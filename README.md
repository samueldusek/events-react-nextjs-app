## To run this app

Clone the project

```bash
  git clone https://github.com/samueldusek/events-react-nextjs-app.git
```

Go to the project directory

```bash
  cd events-react-nextjs-appevents-react-nextjs-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
